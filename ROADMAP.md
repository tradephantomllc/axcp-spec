# AXCP Roadmap

| Milestone | Key features | ETA |
|-----------|--------------|-----|
| **v0.4**  | Capability discovery API, signed bundle exchange | 2025-09 |
| **v0.5**  | Adaptive encryption, remote OTEL streaming       | 2025-12 |
| **v1.0**  | LTS, HIPAA/GDPR compliance, spec freeze          | 2026-03 |

## NEXCP Convergence

AXCP will converge on NEXCP to ensure broader ecosystem compatibility while maintaining core design principles. This convergence will enable seamless interoperability with existing and future agent coordination protocols.

## Open Governance WG

The project will transition to open governance through a dedicated Working Group structure, ensuring transparent decision-making and community-driven development as we approach the v1.0 milestone.
